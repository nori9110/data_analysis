# Phase 1: コア機能実装手順書

## 1. データ基盤構築

### 1.1 プロジェクト環境セットアップ
1. Python仮想環境の作成
   ```bash
   python -m venv venv
   source venv/bin/activate  # macOS/Linux
   ```
2. 必要なパッケージのインストール
   ```bash
   pip install streamlit pandas numpy matplotlib seaborn
   ```
3. requirements.txtの作成
   ```bash
   pip freeze > requirements.txt
   ```

### 1.2 基本的なデータベース構築
1. SQLiteデータベースの作成
2. 基本テーブルスキーマの設計
   - 売上データテーブル
   - 商品マスターテーブル

### 1.3 CSVデータ取り込み機能実装
